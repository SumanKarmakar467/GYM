Place Mixamo-exported GLB workout clips in this folder.

Expected filenames:

- squat.glb
- deadlift.glb
- push-up.glb
- pull-up.glb
- shoulder-press.glb
- curl.glb
- plank.glb
- jumping-jacks.glb
- workout.glb

The workout page automatically maps gym, home, and bodyweight exercises to these files. If a file is missing, the app shows the built-in animated anatomy fallback human with the same 360-degree rotation and muscle heat highlights.
